A plugin to learn about WordPress security concepts through practical examples.

== Description ==

This plugin demonstrates various WordPress security concepts and best practices through a simple form submission system. It includes examples of:

* Form submission handling
* AJAX operations
* Database operations
* Input sanitization
* Output escaping
* Capability checks
* Nonce verification

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/wp-learn-plugin-security`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the shortcode `[wp_learn_form_shortcode]` to display the form
